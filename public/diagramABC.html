Accepted Language = (110)*



A state transition table or state table is a table that shows which state the automaton will move toward when the number of states is finite, based on its current state and based on other inputs. Inputs are usually placed on the left, separated from outputs, which are on the right. Outputs represent the next state of the automaton.
Please note
→: represents the initial state
*: represents the final state
